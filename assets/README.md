## Usage

### In README.md

```markdown
<p align="center">
  <img src="assets/logo.svg" alt="SynaDB Logo" width="150"/>
</p>
```

### In HTML

```html
<img src="assets/logo.svg" alt="SynaDB Logo" width="150">
```

### Favicon

```html
<link rel="icon" type="image/x-icon" href="assets/favicon.ico">
```
